## ğŸ” Authentication Flow

## ğŸ–¼ï¸ Image Upload Flow

1. User selects an image
2. Image is sent using `multipart/form-data`
3. Multer uploads the image to Cloudinary
4. Cloudinary returns a secure URL
5. URL is saved in MongoDB
